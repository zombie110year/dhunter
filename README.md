# 重复猎人

找出指定目录下所有相同的文件.

默认情况下将 md5 值相同的文件归为一组.

# 使用方法

## `root`

第一个参数, 指定搜索起点, 默认为当前工作目录.

```sh
# 搜索当前目录
dhunter

# 搜索 D:/xiaoshipin 目录
dhunter D:/xiaoshipin
```

## `--ignore`

可选参数, 指定文件排除规则, 使用 Python 的 lambda 表达式, 如果文件对象满足此条件, 则不被包含在重复列表中, 传入的参数是名为 `o` 的 `pathlib.Path` 对象. 支持使用正则模块(re).

示例:

```python
o.name !=
```
